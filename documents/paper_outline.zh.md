**备选标题（择一）：**
1. *When, Not What：一个演化的个体潜在状态预测的是人的未来"思考用时"，而非其"选择"*
2. *建模人"何时"行动：演化潜在状态在预测人的未来行为上胜过无记忆的历史条件化*

全文约定：**D** = 演化潜在状态（evolving latent），**B** = 等容量无记忆孪生对照
（equal-capacity memoryless twin）；NLL 越低越好，故 D−B < 0 表示演化潜在状态更优。

P(D−B<0) 是与之并列报告的自助法符号支持度（sign-support），**不是** 频率学派的
p 值。

## 摘要（Abstract）

人不会两次以完全相同的方式行动。刚输一局的棋手，下一局会打得不一样；一名学生在
挣扎着做完十道题后，第十一道的作答方式也会不同。现有对具体个体进行模拟的模型把人
表示为**静态**对象——一个人设（persona）、一个等级分（rating）或一个固定嵌入
（embedding）——因而无法追踪这种会话内的漂移；而已有的历史条件化模型也说不清它们的
增益究竟来自**累积的状态**，还是仅仅来自**看到了近期上下文**。我们用一个随个体自身
"动作+计时"轨迹演化的个体潜在状态来建模这种漂移，并用一个**等容量、同输入的无记忆
孪生对照**在该个体数据的严格未来划分（future split）上评估，从而分离出它的真实价值。
我们的核心发现是一种非对称性：演化状态清晰地体现在人**何时**行动上，而非其**选择什么**
上。在真实的 Lichess 国际象棋数据上，演化潜在状态在 2017–2023 六年跨度、全部八个
"年代×骨干（backbone）"条件下都在预测未来思考用时上胜过孪生对照（差距 −0.021 至
−0.033 nats，每个置信区间都不含零），并且在所测的全部三个 cohort 上仍能在 Allie
之上带来增益——Allie 是一个已发布的思考用时模型，其自身即达到 Spearman 0.62–0.65；
与此相对，走子选择几乎不体现任何状态依赖（一个探针恢复"相对 Maia-2 的偏离"仅
R² = 0.009）。这一计时优势集中在行为最难以用"平均人"预测之处：在方差控制后，时间压力
下高出 2.7–3.6×，对最弱玩家高出约 3×。在八个真实知识追踪（knowledge tracing, KT）
数据集上，演化潜在状态在 24 个"数据集×种子"格中的 22 个上于作答（response）通道
胜出，并且在恢复总体准确率分布上比"平均人"基线接近整整两倍；但在所测的两个教育
数据集上，作答**用时**（response time）并未迁移，这将计时结论的适用范围限定在"时间
被策略性管理"的领域。最后，由哪种通道承载该状态是**依赖骨干的**（backbone-dependent）：
在无语言先验的板面原生（board-native）骨干上，一个训练得到的隐藏向量比一段文字描述
（verbal note）好 0.07–0.12 nats；而在一个指令微调过的 LLM 内部，这一优势消失。

## 1. 引言（骨架）

1. **问题。** 被模拟的人（对手、学生、用户）是针对**具体**个体的模型来评估与训练的；
   具体任务：预测某个指名 Lichess 玩家下一步的思考用时、某个指名学生的下一次作答。
2. **现有方法的失效机理。** 静态的个体嵌入与 cohort 模型把人"冻结"住；历史条件化
   模型把累积状态与近期上下文混为一谈，且据我们所知，没有任何先前工作跑过一个
   容量对齐（capacity-matched）的对照来区分二者（§2）。
3. **洞见。** 如果一个人携带一个演化的行为状态，那么它应当：最清晰地体现在自定步调的
   **计时**上、集中在行为最偏离"平均"之处（时间压力、较弱玩家）、并且需要**累积**才能
   捕获——这三点都可对无记忆孪生对照直接检验。

### 贡献（Contributions）

1. **在严格未来划分上的等容量"演化 vs 无记忆"对照。** 参数、输入、优化器完全相同，
   唯一差别是是否跨步累积状态。演化潜在状态在全部 8 个"年代×骨干"计时条件下胜出
   （表 2），在孪生对照的潜在宽度放大到 2× 时仍然胜出，并且在**锁定**一个已发布的
   思考用时模型**以及**一个稳定的个体标定（per-player calibration）之后仍带来增益——
   由此把个体内部的**动态**贡献从静态个体化（individualization）中分离出来（§4.1、§4.3）。
2. **在真实人类数据上的 when-not-what 非对称性。** 走子选择近乎零效应（最多也依赖
   骨干；"相对 Maia-2 偏离"探针 R² 0.009），而计时优势能挺过等于或高于已发表 SOTA
   秩相关的基线，包括 Allie 已发布的思考用时头（head）（§4.2–4.3，表 3–4）。
3. **一套机理解释。** 计时优势在时间压力下集中 2.7–3.6×（经方差控制）、对最弱玩家
   约 3×；在一个拥有**已知**隐藏状态的合成玩家上，潜在状态能编码它（探针 R² 0.93 vs
   0.65），且钳制（clamp）它会使预测单调变化（§4.4，表 5）。
4. **超出游戏的普适性与实用性。** 在八个真实 KT 数据集上，演化潜在状态在 22/24 个
   种子格上于作答通道胜出；同一潜在状态能恢复真实的总体异质性（Wasserstein 比
   "平均人"基线近 2×），并能生成多样且合理的合成学生（recall 1.00 vs 0.00）。两个真实
   教育数据集的作答用时检验为负，界定了计时结论的适用边界（§4.5–4.6，表 6–7）。
5. **注入通道的优劣次序依赖骨干。** 在板面原生骨干上，训练得到的隐藏向量比文字描述
   好 0.07–0.12 nats；而在 Qwen3 内部，文字描述在所有种子上都能改善计时，隐藏通道
   相对它不再带来任何优势；一个 SFT 探针从 0.6B 到 8B 都复现了"计时 ≫ 走子"
   （§4.7，表 8）。

## 3. 方法（一页）

- **DecisionPoint 接口：** 状态、合法动作、每决策参考量（引擎/IRT 难度）、时间信号、
  近期结果流。国际象棋与 KT 原样共享它。
- **演化潜在状态。** 一个 GRU `f_φ` 累积与对照所见**完全相同**的工程化历史特征
  （时间压力、输棋后、疲劳、势头）；`z_t` 是**因果的**（只依赖 ≤ t 的步）。渲染给骨干时，
  或作为隐藏向量、或作为一段文字描述（一个状态，两条通道）。
- **无记忆孪生对照。** 参数与输入完全相同，但每步把传入的隐藏状态清零。需披露
  `weight_hh` 因构造而失活的细节（≈2/3 的注入器参数因构造获得零梯度）。
- **骨干。** 板面原生的 FEN→起点/落点头（国际象棋）、逻辑回归头（KT）、经 sglang 的
  Qwen3（LLM 分支）。计时头是一个（零膨胀）对数正态，仅从潜在状态读出。
- **训练与评估。** 用 SFT 在每个玩家较早的轨迹上拟合 move-NLL + λ·timing-NLL，损失
  掩码到训练步；在留出的较晚会话上评分；在玩家层面做自助法。

### 4.4 优势落在何处、潜在状态编码了什么？（机理）

| 稳定个体速度基线（van der Linden 类比），5 cohort | 在 2017-04、2021-04（−0.052）、2021-06（−0.137）显著；2019-07、2023-04 为零；全部 5 个点估计偏向 D |
| 合成已知状态探针（留出 R²） | D 0.93 vs B 0.65 |
| 因果钳制 ±ασ | 单调剂量—反应：熵上升、思考用时上升 |
| 走子通道集中性 | 两个 cohort、每个桶都 ≈0（对比项） |

shuffle）对照表明作答通道的优势是与顺序无关的个体化。

  −0.0159 [−0.0202,−0.0118] 但计时 −0.0004 [−0.0059,+0.0059]。假说（标注为假说）：
  教育计时器是附带的 UI 用时，而非被策略性管理的时钟。
- 围棋：一旦控制盘面尺寸，任何尺寸下都为零；残留的 9×9 信号在 2.5× 更大的
  cohort 上塌缩（均值 ≈ 0）。

### 4.6 潜在状态能否恢复并生成一个总体？（实用性）

### 4.7 哪条通道承载状态？（隐藏 vs 文字，按骨干）

**表 8 — 通道对比。**

| 设置 | 对比 | 结果 |
|---|---|---|
| 板面原生，合成 | hidden − verbal | −0.069, P=1.00 |
| 板面原生，真实 2013 | hidden − verbal | −0.117, P=1.00 |
| Qwen3-1.7B LoRA SFT，3 种子 | verbal − none（计时） | −0.0050，3 种子全负 |
| Qwen3-1.7B LoRA SFT，3 种子 | hidden − verbal（计时） | +0.0034（隐藏优势消失） |
| Qwen3 0.6B→8B LoRA SFT | 状态改善计时 Δ | 每个规模都 −0.0107 … −0.0136 |
| Qwen3 4B/8B 全参 SFT | 计时 / 走子 Δ | 计时 −0.0110/−0.0128；走子 −0.0072/−0.0083（LoRA 的走子零是适配器容量假象） |

在无语言先验时，训练得到的隐藏向量是更丰富的通道；而在指令微调过的 LLM 内部，
文字描述承载了状态，隐藏通道相对它不再带来任何优势。在一个稠密 SFT 探针下，
"计时 ≫ 走子"的非对称性在 LLM 中复现，并在全参微调下呈**渐变**（约 1.5×）。
支撑性负结果：冻结的文字注入 ≈ 无关填充文本；稀疏奖励的 RL 学会了任务但无法分辨
状态效应。

## 5. 局限（每条附一个适用边界）

- 头号 D-vs-B 数字用的是从零训练的小骨干；对计时而言这一顾虑因构造（计时头只读
  潜在状态）与 G4 已被消除，但走子上限仍是开放问题。
- 效应量在 −0.01 至 −0.07 nats；其实用价值落在标定（proper 对数似然）与总体
  恢复的演示上，而非原始差距本身。
- when-not-what 非对称性在真实国际象棋与合成 KT 上成立；在所测的两个真实教育
  计时器上不迁移。边界：时间被策略性自定步调的领域。
- "动态 vs 个体化"只有在合成数据上才能完全分离；真实数据上的证据是集中性特征。
- 无真人在线研究；留出未来行为预测与交互式人评回答的是不同问题。
